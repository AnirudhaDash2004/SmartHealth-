# SmartHealth+

SmartHealth+ is an AI-based health risk prediction system using Machine Learning.

## Features
- Health risk prediction
- ML-based analysis
- Data storage using SQLite
- Simple web interface

## Run Project
1. Train model:
   python model_training.py

2. Start app:
   python app.py

3. Open browser:
   http://127.0.0.1:5000/
